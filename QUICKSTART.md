# 🚀 התחלה מהירה - 60 שניות למערכת פעילה!

## הפעלה מהירה

### 1️⃣ הפעל את הסקריפט (פעם אחת)
```bash
./start.sh
```

### 2️⃣ הפעל Backend
```bash
cd backend && source venv/bin/activate && python3 app.py
```

### 3️⃣ הפעל Frontend (טרמינל חדש)
```bash
cd frontend && npm start
```

### 4️⃣ פתח בדפדפן
```
http://localhost:3000
```

## 🎯 שימוש ראשוני

### צור צוות ראשון
1. לחץ על "צוותים" בתפריט
2. "הוסף צוות חדש"
3. מלא: שם, מפקד, חברי צוות
4. שמור!

### הוסף הידרנט
1. לחץ על "הידרנטים"
2. "הוסף הידרנט"
3. מלא פרטים + קואורדינטות
4. עבור ל"מפה" לראות את ההידרנט!

### צור משימה רבעונית
1. לחץ על "משימות"
2. "הוסף משימה"
3. בחר סוג: "רבעונית"
4. בחר רבעון: Q1, Q2, Q3, או Q4
5. הגדר תאריך יעד

## ⚡ טיפים מהירים

### סטטוסים צבעוניים
- 🟢 **ירוק**: תקין / זמין / מוכן
- 🟡 **צהוב**: דורש תחזוקה / בשירות
- 🔴 **אדום**: לא פעיל / דחוף

### סינון משימות
- סנן לפי רבעון (Q1-Q4)
- סנן לפי סטטוס (ממתינה/בביצוע/הושלמה)
- סנן תחזוקה לפי סוג פריט

### עדיפויות משימות

2. 🟡 **גבוהה** - בתוך שבוע

4. ⚪ **נמוכה** - כשיש זמן
