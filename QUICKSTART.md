## התקנה מהירה (5 דקות)

### דרישות מקדימות
- Python 3.8 ומעלה
- Node.js 18 ומעלה
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)

---

## שלב 1: התקנת התלויות

### Backend (Python/Flask)
```bash
cd backend
pip install Flask flask-cors flask-sqlalchemy
```

### Frontend (React)
```bash
cd frontend
npm install
```

---

## שלב 2: הפעלה

### הפעלת Backend (טרמינל 1)
```bash
cd backend
python app.py
```
✅ השרת עולה על http://localhost:5000

### הפעלת Frontend (טרמינל 2)
```bash
cd frontend
npm start
```
✅ הממשק נפתח אוטומטית על http://localhost:3000

---

## שלב 3: שימוש ראשון

1. **פתח את הדפדפן** - http://localhost:3000
2. **לוח הבקרה יטען** - תראה סטטיסטיקות ומפה
3. **הוסף הידרנט ראשון:**
   - לחץ על "הידרנטים" בתפריט
   - לחץ "+ הוסף הידרנט"
   - מלא:
     - מספר סידורי: H-001
     - שם: הידרנט מרכזי
     - מיקום: ליד המועדון
     - קואורדינטות: 31.4117, 34.6667
     - סוג: קרקעי
     - לחץ מים: 5.5 בר
   - שמור

4. **הוסף ארון ציוד:**
   - לחץ על "ארונות ציוד"
   - לחץ "+ הוסף ארון"
   - מלא:
     - מספר ארון: C-001
     - שם: ארון חדר אוכל
     - מיקום: ליד הכניסה הראשית
     - קואורדינטות: 31.4120, 34.6670
   - שמור
   - לחץ "תכולה" והוסף פריטי ציוד:
     - זרנוק 25 מטר (כמות: 2)
     - מטף אבקה 6 ק"ג (כמות: 2, תאריך פג תוקף)

5. **חזור ללוח הבקרה** - תראה את ההידרנט והארון על המפה!

## תכונות עיקריות

### 🗺️ מפה אינטראקטיבית
- הצגת כל ההידרנטים והארונות
- צבעים לפי סטטוס:
  - 🟢 ירוק = תקין
  - 🟡 צהוב = דורש תחזוקה
  - 🔴 אדום = לא תקין / חסר
- לחיצה על סמן = צפייה בפרטים מלאים

### ⚠️ מערכת התראות
המערכת מתריעה אוטומטית על:
- הידרנטים שלא נבדקו 5.5 חודשים
- מטפים שפגים בעוד 30 יום (או כבר פגו)
- משימות שעברו תאריך יעד
- ציוד חסר או דורש החלפה

### 📋 ניהול משימות
- יצירת משימות בדיקה, תחזוקה, תיקון
- הקצאה לחברי צוות
- מעקב עדיפויות (נמוכה → קריטית)
- קישור להידרנט/ארון ספציפי

### 🧰 מעקב מלאי מפורט
- רשימת ציוד מפורטת בכל ארון
- מעקב תאריכי פג תוקף (מטפים)
- התראות אוטומטיות
- סטטוס פריטים: תקין / דורש החלפה / חסר

## טיפים לשימוש

### הזנת קואורדינטות GPS
- השתמש ב-Google Maps:
  1. לחץ ימין על המיקום במפה
  2. לחץ על הקואורדינטות בתפריט
  3. הקואורדינטות יועתקו (לדוגמה: 31.4117, 34.6667)
  4. הדבק בשדות latitude ו-longitude

### ניהול יומיומי
1. **בוקר:**
   - בדוק לוח בקרה להתראות
   - סקור משימות היום
   
2. **אחר פעילות:**
   - עדכן סטטוס משימות
   - תעד תחזוקה שבוצעה
   
3. **תקופתי:**
   - בדוק הידרנטים (כל 6 חודשים)
   - עדכן תאריכי פג תוקף מטפים
   - סקור ארונות ציוד

### גיבוי נתונים
מסד הנתונים נמצא ב:
```
backend/fire_department.db
```
מומלץ לגבות קובץ זה באופן קבוע.

## פתרון בעיות

### Backend לא עולה
```bash
# ודא שכל הספריות מותקנות:
```

  - Latitude: 29-33
  - Longitude: 34-36
