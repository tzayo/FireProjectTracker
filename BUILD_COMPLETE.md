## 🛠️ טכנולוגיות

### Backend
- **Python 3** + **Flask** - שרת REST API
- **SQLAlchemy** - ORM
- **SQLite** - מסד נתונים
- **Flask-CORS** - תמיכה ב-CORS

### Frontend
- **React 18** - ספריית UI מודרנית
- **React Router** - ניתוב
- **Axios** - קריאות HTTP
- **Leaflet** - מפות אינטראקטיביות
- **Tailwind CSS** - עיצוב מודרני
- **RTL Support** - תמיכה מלאה בעברית

## 🚀 איך להתחיל? (3 צעדים!)

### שלב 1: התקנה (פעם אחת)
```bash
./start.sh
```

הסקריפט יטפל בכל ההתקנות אוטומטית!

### שלב 2: הפעל Backend
```bash
cd backend
source venv/bin/activate
python3 app.py
```

✅ השרת רץ על: **http://localhost:5000**

### שלב 3: הפעל Frontend (טרמינל נוסף)
```bash
cd frontend
npm start
```

✅ האפליקציה נפתחת ב: **http://localhost:3000**
