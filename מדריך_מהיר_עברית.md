# 🚒 מערכת ניהול כיבוי אש - מדריך מהיר בעברית

---

## 🎯 מה זאת המערכת?

מערכת מקיפה לניהול כל תשתיות כיבוי האש בקיבוץ:
- מעקב אחר כל ברזי הכיבוי (הידרנטים)
- ניהול ארונות ציוד ומלאי
- מעקב תחזוקה ובדיקות תקופתיות
- התראות אוטומטיות
- מפות אינטראקטיביות

---

## ⚡ התקנה והפעלה מהירה

### מה צריך לפני שמתחילים?
- מחשב עם Windows/Mac/Linux
- חיבור לאינטרנט (רק להתקנה)

### התקנה (פעם אחת)

#### שלב 1: הורד את הקוד
הקוד כבר נמצא בתיקייה `/workspace`

#### שלב 2: התקן תוכנות נדרשות

**Python:**
- הורד מ-https://www.python.org/downloads/
- התקן (לחץ "Add to PATH" בהתקנה!)

**Node.js:**
- הורד מ-https://nodejs.org/
- בחר בגרסת LTS
- התקן

#### שלב 3: התקן את המערכת

**אפשרות 1: באמצעות סקריפט אוטומטי (מומלץ)**
```bash
./start-system.sh
```

**אפשרות 2: ידנית**

פתח 2 חלונות Terminal:

**Terminal 1 - Backend:**
```bash
cd backend
pip install Flask flask-cors flask-sqlalchemy
python app.py
```
→ השרת עולה על http://localhost:5000

**Terminal 2 - Frontend:**
```bash
cd frontend
npm install
npm start
```
→ הממשק נפתח על http://localhost:3000

## 🖥️ שימוש ראשון - צעד אחר צעד

### 1. פתח את המערכת
פתח דפדפן וגש ל: **http://localhost:3000**

תראה את לוח הבקרה עם 8 תפריטים:
- 🏠 לוח בקרה
- 👥 צוותים
- 🚰 הידרנטים
- 🧰 ארונות ציוד
- ✓ משימות
- 🔧 תחזוקה
- 👤 מתנדבים
- 📋 פעילויות

### 2. הוסף הידרנט ראשון

1. לחץ על **"הידרנטים"** בתפריט העליון
2. לחץ **"+ הוסף הידרנט"**
3. מלא את הפרטים:

```
מספר סידורי: H-001
שם: הידרנט מרכזי
מיקום: ליד המועדון
סוג: קרקעי
לחץ מים: 5.5 (בר)
קוטר: 4 (אינץ')
קו רוחב: 31.4117
קו אורך: 34.6667
סטטוס: תקין
```

4. לחץ **"הוסף"**

✅ ההידרנט נוסף! תראה אותו ברשימה ובמפה.

### 3. מצא קואורדינטות GPS

**באמצעות Google Maps:**
1. פתח Google Maps
2. חפש את המיקום או לחץ על המפה
3. לחץ ימין על הנקודה
4. לחץ על הקואורדינטות בתפריט (הן יועתקו)
5. הדבק במערכת

דוגמה: `31.4117, 34.6667`
- 31.4117 = Latitude (קו רוחב)
- 34.6667 = Longitude (קו אורך)

### 4. הוסף ארון ציוד

1. לחץ על **"ארונות ציוד"**
2. לחץ **"+ הוסף ארון"**
3. מלא:

```
מספר ארון: C-001
שם: ארון חדר אוכל
מיקום: ליד הכניסה הראשית לחדר האוכל
סוג ארון: סטנדרטי
קו רוחב: 31.4120
קו אורך: 34.6670
סטטוס: תקין
```

4. לחץ **"הוסף"**

### 5. הוסף ציוד לארון

1. ברשימת הארונות, לחץ **"תכולה"** על הארון שהוספת
2. לחץ **"+ הוסף פריט"**
3. הוסף זרנוק:

```
סוג פריט: זרנוק
שם: זרנוק 25 מטר
כמות: 2
אורך: 25 (מטרים)
סטטוס: תקין
```

4. הוסף מטף:

```
סוג פריט: מטף
שם: מטף אבקה 6 ק"ג
כמות: 2
תאריך פג תוקף: [בחר תאריך בעוד שנה]
סטטוס: תקין
```

**חשוב!** הזן תאריך פג תוקף כדי לקבל התראות!

### 6. צפה בלוח הבקרה

1. לחץ **"לוח בקרה"** בתפריט
2. תראה:
   - סטטיסטיקות (כמה הידרנטים, כמה ארונות)
   - מפה עם כל הנכסים
   - התראות (אם יש)

## 🗺️ שימוש במפות

### צבעים
- 🟢 **ירוק** = הכל בסדר, תקין
- 🟡 **צהוב** = דורש תשומת לב, תחזוקה
- 🔴 **אדום** = בעיה דורשת טיפול מיידי

### לחיצה על סמן
- לחץ על נקודה במפה (הידרנט או ארון)
- יפתח חלון עם כל הפרטים
- תוכל לערוך או לצפות במידע נוסף

## ⚠️ התראות

המערכת מתריעה אוטומטית על:

### 🚰 הידרנטים שטעונים בדיקה
אם הידרנט לא נבדק **5.5 חודשים** - תקבל התראה

### 🧯 מטפים שפגים
- **30 ימים לפני פג תוקף** - התראה צהובה
- **פג תוקף עבר** - התראה אדומה

### ✓ משימות שעברו דדליין
משימות שלא הושלמו עד התאריך - התראה

**איפה רואים את ההתראות?**
- בלוח הבקרה - פאנל עליון אדום/צהוב
- בכל רענון של הדף

## 📋 ניהול משימות

### יצירת משימה

1. לחץ **"משימות"**
2. לחץ **"+ משימה חדשה"**
3. מלא:

```
כותרת: בדיקת הידרנט H-001
סוג: בדיקה
עדיפות: בינונית
קשור להידרנט: H-001
תאריך יעד: [בחר תאריך]
אחראי: [שם חבר צוות]
```

4. לחץ **"צור משימה"**

### עדכון משימה

1. לחץ על המשימה ברשימה
2. שנה סטטוס:
   - חדש → בטיפול (כשמתחילים)
   - בטיפול → הושלם (כשגמרו)
3. הוסף הערות על מה שנעשה

---

## 🔧 תחזוקה שוטפת

### לאחר בדיקת הידרנט:

1. לחץ **"הידרנטים"**
2. מצא את ההידרנט שנבדק
3. לחץ **"ערוך"**
4. עדכן:
   - סטטוס (אם יש בעיה)
   - לחץ מים (אם נמדד)
   - הוסף הערה: "נבדק ב-[תאריך] - הכל תקין"
5. שמור

### לאחר החלפת ציוד:

1. לחץ **"ארונות ציוד"**
2. לחץ **"תכולה"** על הארון הרלוונטי
3. מצא את הפריט שהוחלף
4. לחץ **"ערוך"**
5. עדכן תאריך פג תוקף (אם מטף)
6. עדכן סטטוס
7. שמור

## 📞 שאלות נפוצות

### איך מוצאים קואורדינטות GPS?

**Google Maps:**
1. פתח Google Maps
2. לחץ ימין על הנקודה
3. לחץ על המספרים שמופיעים (לדוגמה: 31.4117, 34.6667)
4. הם מועתקים אוטומטית
5. הדבק במערכת

### מה אם טעיתי במספר סידורי?

מספר סידורי **לא ניתן** לשינוי לאחר יצירה (כדי למנוע בלבול).

**פתרון:** מחק את הפריט וצור מחדש עם המספר הנכון.

### איך מוסיפים תמונות?

כרגע המערכת תומכת ב-URL של תמונות.
העלה תמונה לכונן משותף והדבק קישור.

**בעתיד:** העלאה ישירה למערכת.

### המפה לא נטענת

- בדוק חיבור אינטרנט (המפות נטענות מהרשת)
- נסה לרענן את הדף
- אם הבעיה נמשכת: השתמש בתצוגת רשימה

### איך עוצרים את המערכת?

**אפשרות 1:**
```bash
./stop-system.sh
```

**אפשרות 2:**
לחץ `Ctrl+C` בכל אחד מחלונות ה-Terminal

## 🎯 מקרי שימוש נפוצים

### תרחיש 1: בדיקה תקופתית של הידרנט

1. היכנס למערכת
2. לחץ "הידרנטים"
3. מצא את ההידרנט (חפש לפי מספר או שם)
4. לחץ "ערוך"
5. עדכן:
   - לחץ מים (אם נמדד)
   - סטטוס
   - הערות: "נבדק ב-[תאריך] - תקין"
6. שמור
7. (אופציונלי) צור משימה לבדיקה הבאה בעוד 6 חודשים

### תרחיש 2: החלפת מטף שפג תוקפו

1. המערכת מתריעה: "מטף X בארון C-003 פג תוקף"
2. לחץ "ארונות ציוד"
3. מצא ארון C-003
4. לחץ "תכולה"
5. מצא את המטף שפג
6. לחץ "מחק" (אם הוצא לגמרי)
   או "ערוך" ועדכן תאריך פג תוקף חדש
7. אם הוספת מטף חדש, לחץ "+ הוסף פריט"

### תרחיש 3: דיווח על בעיה בארון

1. חבר צוות מגלה שחסר זרנוק
2. נכנס למערכת
3. "ארונות ציוד" → מצא את הארון
4. "תכולה"
5. מצא את הזרנוק
6. לחץ "ערוך"
7. שנה סטטוס ל-"חסר"
8. שמור

→ המערכת תתריע אוטומטית שחסר ציוד בארון!
