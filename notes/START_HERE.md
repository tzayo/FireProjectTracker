## ⚡ התקנה ראשונה?

### דרישות
- Python 3.8+
- Node.js 18+
- 5 דקות זמן

### צעדים

```bash
# 1. Backend
cd backend
pip install -r requirements.txt

# 2. Frontend
cd ../frontend
npm install

# 3. הפעל
cd ..
./start-system.sh
```

## 🗺️ מפות אינטראקטיביות

המערכת כוללת 3 מפות:

1. **לוח הבקרה** - כל הנכסים ביחד
2. **הידרנטים** - פוקוס על ברזים
3. **ארונות** - פוקוס על ציוד

**צבעים:**
- 🟢 ירוק = תקין
- 🟡 צהוב = דורש תשומת לב
- 🔴 אדום = בעיה
